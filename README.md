# Zerra

Zerra is a lightweight micro-posting platform designed for modern web and mobile applications.

The project explores the architecture behind real-time social networks such as X (Twitter), focusing on scalability, performance, and clean system design.

Zerra allows users to create posts, follow other users, interact with content, and explore a dynamic timeline of conversations.

This project is built as a learning platform for understanding how social networks are designed and implemented.

## Features

- User authentication
- Create and delete posts
- Follow and unfollow users
- Personalized timeline feed
- Likes and replies
- User profiles
- Scalable backend architecture
